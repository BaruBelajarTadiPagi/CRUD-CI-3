Petunjuk Instalasi project
1. Extract project 'Interview-Web'
2. Download aplikasi Xampp, dengan php versi PHP version: 7.4.23
	(pastikan hal ini sesuai, karena CodeIgniter3 hanya bisa dijalankan dengan php versi 5-7)
3. Letakkan folder project kedalam folder D:/xammpp/htdocs/{{ letakkan disini }}
4. Jalankan aplikasi Xampp nyalakan Apache dan MySQL
5. Buka browser chrome dan login ke http://localhost/phpmyadmin/
6. 'apabila sudah masuk halaman utama phpmyadmin' , pilih menu "Import" pada menu yang ada di atas.
7. Pilih File yang sudah di siapkan >> interview-web.sql <<, kemudian jalankan 
8. apabila sudah berhasil melakukan import file .sql.
	buka folder project 'Interview-Web' dengan Visual Studio Code (disarankan) atau Text Editor lainnya
9. Buka file yang berada di 'application/config/config.php'.
	cari code > $config['base_url']
	pada file tersebut, anda bisa mengubah lokasi filenya sesuai dengan yang ada di laptop saat ini.
	Pastikan foldernya sama dan juga menggunakan http://localhost/{{ lokasi foldernya }}
	(apabila salah penulisan dapat menjadi masalah saat menjalankan program)
10. Buka browser chrome, dan copy paste dari file 'application/config/config.php' pada code >> $config['base_url'] << (lihat nomor 9)
	Aplikasi dapat dijalankan.
